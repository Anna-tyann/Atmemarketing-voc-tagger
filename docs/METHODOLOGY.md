# VOC 分析方法论

> 基于 vocovoca.com 16维分析框架扩展

---

## 数据处理流程

```
Amazon 评论数据 
  ↓
Excel 加载 & 列验证
  ↓
18维标签提取（10产品 + 8行为场景）
  ↓
评分主导 + 语义微调情感分析
  ↓
L1/L2 统计聚合 & 洞察自动生成
  ↓
交互式 HTML 报告
```

---

## 18维分析框架

### 产品功能维度（10个）

| 维度 | 分析内容 | 适用场景 |
|------|---------|---------|
| 核心功能 | 产品使用效果 | 产品迭代、R&D |
| 性能表现 | 动力/效率/速度 | 产品优化 |
| 产品设计与做工 | 外观/材质/尺寸 | 工业设计 |
| 便携与出行 | 携带/移动便利性 | 产品定位 |
| 电池与续航 | 电池/充电体验 | 硬件升级 |
| 易用性与操作 | 上手难度/交互 | UX 优化 |
| 价格与价值 | 性价比感知 | 定价策略 |
| 可靠性与售后 | 故障率/质保 | 品控/客服 |
| 品牌与推荐 | 品牌信任/NPS | 品牌运营 |
| 整体评价 | 总体满意度 | 竞争定位 |

### 行为场景维度（8个）

| 维度 | 核心问题 | 营销应用 |
|------|---------|---------|
| 🎯 购买动因 | 用户为什么买？ | 转化文案、投放定向 |
| 📍 使用地点 | 用户在哪用？ | 场景化素材、网站视觉 |
| ⏰ 使用场景 | 用户什么时候用？ | 内容策略、使用时机 |
| 👤 用户画像 | 谁在用？ | 精准投放、人群包 |
| 💡 用户兴趣 | 用户还喜欢什么？ | 网红合作、跨界营销 |
| 🏷️ 品牌/竞品对比 | 用户对比谁？ | 竞品截流、差异化 |
| 🔮 未满足需求 | 用户还想要什么？ | 产品创新方向 |
| 🔄 复购/忠诚行为 | 用户会再买/推荐吗？ | CRM策略 |

## 核心指标体系

| 指标 | 含义 | 计算 |
|------|------|------|
| **声量 (Mention)** | 该话题被提及次数 | 绝对数量 |
| **正面率 (Positive Rate)** | 正面评价占比 | 正面数/总数×100% |
| **负面率 (Negative Rate)** | 负面评价占比 | 负面数/总数×100% |
| **紧急度 (Urgency)** | 改进优先级 | 声量×(100-正面率) |

---

## 情感分析原理

```
评分 ≥ 4 → 默认正面（仅当关键词紧邻否定词时翻转）
评分 ≤ 2 → 默认负面（仅当关键词紧邻强正面词时翻转）
评分 = 3 → 中性（根据关键词上下文判断）

特殊规则："no leak" → 正面（否定负面词=正面含义）
```

## 可视化策略

### 散点图
- X轴：声量（关注度）
- Y轴：正面率（满意度）
- 气泡大小：提及频次
- 用途：识别高声量高满意度（优势）和低声量低满意度（痛点）

### 堆叠柱状图
- 展示 L1 维度的正面/负面/中性分布
- 用途：快速定位满意度高低区

### 洞察卡片
- 痛点：红色左边框，粉色背景
- 优势：绿色左边框，浅绿背景
- 每张卡片含：标题、统计描述、3条原文引用

## 参考线标准

| 参考线 | 含义 |
|--------|------|
| 50% 满意度分界线（红色虚线） | 低于此线需重点关注 |
| 高关注度线（蓝色虚线） | 提及次数超过此值说明为用户核心关注点 |

---

## 报告质量标准

- ✅ 每评论至少 1 个标签
- ✅ 正面率与均分偏差 < 20%
- ✅ 5-15 个 L1 类别
- ✅ 至少 3 个痛点 + 3 个优势
- ✅ 所有洞察附原文引用
